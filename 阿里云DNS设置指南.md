# 阿里云OSS域名绑定和DNS设置指南

## 🎯 当前状态
您已登录到阿里云控制台：`https://oss.console.aliyun.com/overview`

## 🔧 第一步：在OSS控制台绑定域名

### 操作步骤：
1. **点击左侧菜单** → **对象存储OSS**
2. **选择存储桶**：`financial-tax-website`（如不存在请先创建）
3. **进入存储桶详情页**
4. **点击左侧菜单** → **传输管理** → **域名管理**
5. **点击"绑定用户域名"**
6. **输入域名**：`www.0slm.com.cn`
7. **选择"自动添加CNAME记录"**（推荐）
8. **点击"提交"**

### 验证域名所有权：
- 阿里云会显示**TXT记录**或**文件验证**方式
- 按照提示在域名注册商处添加TXT记录
- 等待验证通过（通常几分钟）

## 🔗 第二步：设置DNS记录

### 如果您的域名在阿里云万网：
1. **登录阿里云控制台** → **域名** → **域名列表**
2. **找到域名**：`0slm.com.cn`
3. **点击"解析"**
4. **添加CNAME记录**：

```
记录类型：CNAME
主机记录：www
记录值：financial-tax-website.oss-cn-shenzhen.aliyuncs.com
TTL：10分钟
```

5. **保存设置**

### 如果您的域名在其他注册商：
1. **登录域名注册商管理后台**
2. **找到DNS管理/域名解析**
3. **添加CNAME记录**：

```
名称/主机：www
类型：CNAME
值/目标：financial-tax-website.oss-cn-shenzhen.aliyuncs.com
TTL：600（10分钟）
```

## 🌐 常见域名注册商DNS设置位置

### 阿里云万网：
- 控制台 → 域名 → 域名列表 → 点击域名 → 解析设置

### 腾讯云DNSPod：
- 控制台 → 域名注册 → 我的域名 → 点击域名 → 解析

### 新网：
- 会员中心 → 域名管理 → 点击域名 → 域名解析

### 西部数码：
- 会员中心 → 域名管理 → 点击域名 → 域名解析

## ⏱️ 生效时间
- **DNS生效**：2-48小时（通常10-30分钟）
- **HTTPS证书**：自动申请，几分钟到几小时
- **备案验证**：立即生效

## ✅ 验证绑定是否成功

### 方法一：DNS查询
```bash
nslookup www.0slm.com.cn
```
应该显示指向阿里云OSS的CNAME记录

### 方法二：直接访问
在浏览器访问：`http://www.0slm.com.cn`
应该显示您的网站内容

### 方法三：OSS控制台检查
在OSS域名管理页面查看绑定状态
