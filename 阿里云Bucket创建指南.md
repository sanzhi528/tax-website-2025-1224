# 阿里云OSS Bucket创建详细指南

## 🎯 当前操作步骤
您需要先创建Bucket，然后再绑定域名。

## 🔧 第一步：创建Bucket

### 操作位置：
- **当前页面**：`https://oss.console.aliyun.com/overview`
- **操作**：点击左侧菜单 → "对象存储OSS" → 点击"创建Bucket"

### Bucket配置参数：

#### 1. 基本配置
```
Bucket名称：financial-tax-website
地域：华东1（杭州）[推荐]
   - 或者选择：华南1（深圳）
存储类型：标准存储
```

#### 2. 高级配置（重要）
```
读写权限：公共读 [必须选择]
版本控制：关闭
服务端加密：关闭
同城冗余存储：关闭
```

#### 3. 确认创建
- 阅读并同意服务条款
- 点击"确定"创建Bucket

## 📤 第二步：上传网站文件

### 上传方法一：控制台上传
1. **进入刚创建的Bucket**：`financial-tax-website`
2. **点击"上传文件"**
3. **选择文件**：
   - 选择本地文件夹：`e:\20251124\网站建设`
   - 选中所有文件：`index.html`, `style.css`, `script.js`, `images/`等
4. **设置权限**：选择"公共读"
5. **点击上传**

### 上传方法二：使用OSS Browser（推荐批量上传）
1. **下载OSS Browser工具**
2. **使用AccessKey登录**
3. **拖拽整个网站文件夹上传**

## 🌐 第三步：配置静态网站托管

### 配置步骤：
1. **在Bucket详情页** → **"基础设置"**
2. **找到"静态页面"配置**
3. **开启静态网站托管**
4. **设置参数**：
   ```
   默认首页：index.html
   默认404页：index.html [单页应用需要]
   子目录首页：关闭
   ```
5. **保存配置**

## 🔗 第四步：绑定域名

### 绑定步骤：
1. **在Bucket详情页** → **"传输管理"** → **"域名管理"**
2. **点击"绑定用户域名"**
3. **填写域名信息**：
   ```
   域名：www.0slm.com.cn
   自动添加CNAME记录：开启 [推荐]
   ```
4. **点击"提交"**

### 域名验证：
- 阿里云会要求验证域名所有权
- 按照提示在DNSPod添加TXT记录
- 等待验证通过（通常几分钟）

## ✅ 验证创建成功

### 验证方法一：访问测试地址
创建Bucket后，您会获得默认访问地址：
```
http://financial-tax-website.oss-cn-hangzhou.aliyuncs.com
```
访问这个地址应该显示您的网站首页。

### 验证方法二：文件列表检查
在Bucket文件管理页面，应该看到：
- index.html
- style.css  
- script.js
- images/文件夹
- 其他网站文件

## 💡 重要注意事项

### 1. Bucket名称规则
- 必须全局唯一
- 只能包含小写字母、数字、短横线
- 长度3-63字符
- 不能以短横线开头或结尾

### 2. 地域选择建议
- **华东1（杭州）**：访问速度均衡，推荐选择
- **华南1（深圳）**：南方用户访问快
- **华北2（北京）**：北方用户访问快

### 3. 权限设置
- **必须选择"公共读"**，否则网站无法访问
- 上传文件时也要选择"公共读"权限

## 🚨 常见问题解决

### 问题1：Bucket名称已存在
**解决**：换个名称，如：`financial-tax-website-001`

### 问题2：上传文件失败
**解决**：检查文件大小限制（单个文件最大5GB）

### 问题3：网站访问显示403错误
**解决**：检查Bucket和文件权限是否设置为"公共读"

### 问题4：静态网站配置不生效
**解决**：等待几分钟，清除浏览器缓存重新访问
