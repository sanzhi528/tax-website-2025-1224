# 财务税务代理公司网站

这是一个专业的财务税务代理公司网站，提供工商代理、知识产权、会计服务和银行开户等一站式企业服务。

## 功能特性

### 前台功能
- **首页展示**：企业介绍、服务展示、公司优势
- **服务详情**：详细的服务内容、价格、所需材料
- **在线咨询**：客户咨询表单，支持多种业务类型
- **用户注册/登录**：客户账户管理
- **响应式设计**：适配各种设备屏幕

### 后台管理系统
- **仪表板**：数据统计和快速操作
- **公司信息管理**：基本信息、LOGO、社交媒体
- **服务管理**：服务列表、添加、编辑、删除
- **咨询管理**：客户咨询处理、状态更新
- **用户管理**：客户信息管理、VIP状态
- **内容编辑**：网站内容管理、文章发布、FAQ
- **系统设置**：基本配置、邮件设置、系统维护

## 技术栈

- **前端**：HTML5, CSS3, JavaScript (ES6+)
- **样式**：自定义CSS + Font Awesome图标
- **数据存储**：LocalStorage (前端模拟)
- **响应式**：CSS Grid + Flexbox

## 快速开始

### 方法一：直接运行（推荐）
1. 双击打开 `index.html` 文件即可查看前台网站
2. 双击打开 `admin/login.html` 文件即可进入后台管理
3. 无需安装任何依赖，立即体验完整功能

### 方法二：使用本地服务器（可选）
如需API功能（用户注册、咨询提交等）：
1. 确保已安装 Node.js (版本 14.0.0 或更高)
2. 安装依赖: `npm install`
3. 启动服务器: `npm start`
4. 访问 http://localhost:3000

## 部署说明

### 开发环境部署
1. 克隆项目到本地
2. 安装依赖: `npm install`
3. 启动开发服务器: `npm start`

1. 将项目文件上传到Web服务器
2. 配置Web服务器指向项目根目录
3. 确保所有静态资源可访问
4. 配置域名和SSL证书

### 数据库部署（可选）
如需完整的API功能：

2. 导入数据库设计文档中的SQL语句

4. 启动API服务器
