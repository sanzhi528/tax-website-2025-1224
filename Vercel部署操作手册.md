# Vercel部署操作手册 - 财务税务代理公司网站

## 🎯 部署目标
将您的网站免费部署到Vercel平台，并绑定域名实现互联网访问。

## 📋 部署前准备

### 1. 确认网站文件完整
您的网站包含以下关键文件：
- ✅ `index.html` - 首页
- ✅ `css/style.css` - 样式文件
- ✅ `js/main.js` - 脚本文件
- ✅ `admin/` - 后台管理页面
- ✅ `vercel.json` - 部署配置文件

### 2. 准备域名信息
您需要绑定的域名：
- 主域名：www.0slm.com
- 备用域名：0slm.com.cn, bentiun.com等

## 🚀 部署步骤详解

### 第一步：注册Vercel账户

1. **打开浏览器**，访问：https://vercel.com
2. **点击"Sign Up"** 注册账户
3. **推荐使用GitHub账户注册**（更方便后续管理）
   - 如果没有GitHub账户，可以用邮箱注册
4. **完成邮箱验证**

### 第二步：创建新项目

1. **登录Vercel控制台**
2. **点击"New Project"**（新建项目）
3. **选择"Import Git Repository"**
   - 由于您没有GitHub仓库，选择"Deploy Manually"（手动部署）

### 第三步：上传网站文件

**方法一：拖拽上传（推荐）**
1. 将整个`网站建设`文件夹拖拽到Vercel上传区域
2. 或者点击"Browse"选择文件夹

**方法二：压缩包上传**
1. 右键点击`网站建设`文件夹
2. 选择"发送到" → "压缩(zipped)文件夹"
3. 将生成的zip文件上传到Vercel

### 第四步：配置部署设置

1. **项目名称**：输入 `financial-tax-website`
2. **框架预设**：选择 `Other`（其他）
3. **根目录**：保持默认（会自动识别）
4. **点击"Deploy"** 开始部署

### 第五步：等待部署完成

- 部署过程通常需要1-3分钟
- 部署完成后会显示：
  - ✅ 部署成功
  - 🔗 临时访问地址（如：`financial-tax-website.vercel.app`）

## 🔗 绑定自定义域名

### 第一步：在Vercel添加域名

1. 进入项目 → **Settings** → **Domains**
2. 点击 **"Add"** 添加域名
3. 输入：`www.0slm.com`
4. 点击 **"Add"** 确认

### 第二步：在域名注册商设置DNS

**以阿里云为例：**
1. 登录阿里云控制台
2. 进入**域名管理** → **域名解析**
3. 找到`0slm.com`域名
4. 添加以下DNS记录：

```
记录类型：CNAME
主机记录：www
记录值：cname.vercel-dns.com
TTL：10分钟

记录类型：A
主机记录：@
记录值：76.76.21.21
TTL：10分钟
```

### 第三步：验证域名绑定

1. 返回Vercel的Domains页面
2. 等待状态变为 **"Valid"**（有效）
3. 这个过程可能需要几分钟到几小时

## 🌐 测试访问

### 临时地址（立即生效）
- https://financial-tax-website.vercel.app

### 自定义域名（DNS生效后）
- https://www.0slm.com

## ⚡ 多域名设置（可选）

如果您想绑定其他域名：

1. 在Vercel的Domains页面继续添加：
   - www.0slm.com.cn
   - www.bentiun.com
   - 等其他域名

2. 在每个域名的注册商处设置相同的DNS记录

## 🔧 常见问题解决

### 问题1：DNS解析不生效
- **原因**：DNS缓存或设置错误
- **解决**：等待2-48小时，或联系域名注册商客服

### 问题2：网站显示404错误
- **原因**：文件路径错误
- **解决**：检查`vercel.json`配置是否正确

### 问题3：HTTPS证书问题
- **原因**：Vercel自动配置需要时间
- **解决**：等待证书自动签发（通常几分钟）
