# 财务税务代理公司网站 - 启动说明

## 项目概述

这是一个完整的财务税务代理公司网站，包含前台展示和后台管理系统。项目采用纯前端技术栈，无需服务器即可运行。

## 快速启动

### 方法一：直接打开HTML文件（推荐）

1. 在文件管理器中找到项目根目录
2. 双击打开以下文件即可：
   - **前台网站**: `index.html`
   - **后台登录**: `admin/login.html`

### 方法二：使用本地服务器（可选）

如果需要API功能（如用户注册、咨询提交等），可以启动本地服务器：

1. 确保已安装 Node.js (版本 14.0.0 或更高)
2. 在项目根目录打开命令行/终端
3. 运行以下命令安装依赖：
   ```bash
   npm install
   ```
4. 启动服务器：
   ```bash
   npm start
   ```
5. 在浏览器中访问：
   - 前台网站: http://localhost:3000/index.html
   - 后台管理: http://localhost:3000/admin/login.html

## 注意事项

1. **数据存储**: 当前使用本地存储(localStorage)保存数据，刷新页面后数据会重置
2. **API功能**: 如需完整的API功能，请启动本地服务器
3. **文件路径**: 确保所有HTML文件在同一目录下，避免路径问题
4. **安全提示**: 演示版本使用简单验证，生产环境请加强安全措施
