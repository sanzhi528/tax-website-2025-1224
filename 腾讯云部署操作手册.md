## 🚀 部署步骤详解

### 第一步：注册腾讯云账户

1. **访问官网**：https://cloud.tencent.com
2. **点击"免费注册"**
3. **选择注册方式**：
   - 微信扫码注册（最方便）
   - 手机号注册
   - 邮箱注册
4. **完成实名认证**（企业或个人认证）

### 第二步：开通静态网站托管服务

1. **登录腾讯云控制台**
2. **搜索"静态网站托管"**
3. **点击"立即使用"**
4. **选择"免费套餐"**
5. **阅读并同意服务协议**
6. **点击"开通"**

### 第三步：创建存储桶（Bucket）

1. **进入静态网站托管控制台**
2. **点击"创建存储桶"**
3. **填写存储桶信息**：
   - 存储桶名称：`financial-tax-website`
   - 所属地域：选择离您最近的（如：华南-广州）
   - 访问权限：**公有读私有写**（重要！）
4. **点击"确定"创建**

### 第四步：上传网站文件

**方法一：控制台直接上传**
1. 进入刚创建的存储桶
2. 点击"上传文件"
3. 选择`网站建设`文件夹中的所有文件
4. 点击"开始上传"

**方法二：使用COS Browser工具**
1. 下载腾讯云COS Browser客户端
2. 登录您的账户
3. 拖拽文件夹上传（更快速）

### 第五步：配置静态网站

1. **进入存储桶 → 基础配置**
2. **找到"静态网站"配置**
3. **开启静态网站功能**
4. **设置索引文档**：`index.html`
5. **设置错误文档**：`index.html`（单页应用）
6. **保存配置**

### 第六步：获取访问地址

配置完成后，您会获得：
- **临时访问地址**：`https://financial-tax-website-125xxxxxx.cos-website.ap-guangzhou.myqcloud.com`
- 这个地址可以立即访问您的网站

## 🔗 绑定自定义域名

### 第一步：在腾讯云添加域名

1. **进入存储桶 → 域名管理**
2. **点击"添加域名"**
3. **输入您的域名**：`www.0slm.com`
4. **选择"开启CDN加速"**（推荐）
5. **点击"保存"**

### 第二步：在域名注册商设置DNS

**以阿里云为例：**
1. 登录域名注册商控制台
2. 进入**域名解析**设置
3. 添加CNAME记录：

```
记录类型：CNAME
主机记录：www
记录值：cdn地址（腾讯云提供）
TTL：600
```

### 第三步：等待生效

- DNS生效时间：几分钟到几小时
- 生效后访问：`https://www.0slm.com`

## 🔧 常见问题解决

### 问题1：网站无法访问
- **检查**：存储桶权限是否为"公有读"
- **解决**：修改存储桶权限设置

### 问题2：HTTPS证书问题
- **检查**：域名是否已备案
- **解决**：腾讯云自动签发SSL证书

### 问题3：文件更新不生效
- **检查**：CDN缓存设置
- **解决**：刷新CDN缓存或等待缓存过期
