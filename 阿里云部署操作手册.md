## 🚀 部署步骤详解

### 第一步：登录阿里云控制台
1. **访问**：https://www.aliyun.com
2. **登录**您的阿里云账户
3. **进入控制台**

### 第二步：开通OSS服务
1. **搜索"对象存储OSS"**
2. **点击"立即开通"**
3. **选择"按量付费"**（实际费用很低）
4. **阅读并同意协议**

### 第三步：创建存储桶（Bucket）
1. **进入OSS控制台**
2. **点击"创建Bucket"**
3. **填写配置信息**：
   ```
   Bucket名称：financial-tax-website
   地域：华东1（杭州）或华南1（深圳）
   存储类型：标准存储
   读写权限：公共读
   ```
4. **点击"确定"创建**

### 第四步：上传网站文件
**方法一：控制台上传**
1. 进入刚创建的Bucket
2. 点击"上传文件"
3. 选择`网站建设`文件夹中的所有文件
4. 设置文件权限为"公共读"
5. 点击上传

**方法二：使用OSS Browser工具**
1. 下载OSS Browser客户端
2. 使用AccessKey登录
3. 拖拽文件夹批量上传

### 第五步：配置静态网站
1. **进入Bucket → 基础设置**
2. **找到"静态页面"配置**
3. **开启静态网站托管**
4. **设置默认首页**：`index.html`
5. **设置默认404页**：`index.html`
6. **保存配置**

### 第六步：获取访问地址
配置完成后，您会获得：
- **OSS访问域名**：`financial-tax-website.oss-cn-hangzhou.aliyuncs.com`
- **静态网站访问地址**：`financial-tax-website.oss-cn-hangzhou.aliyuncs.com`

## 🔗 绑定自定义域名

### 第一步：在OSS绑定域名
1. **进入Bucket → 传输管理 → 域名管理**
2. **点击"绑定用户域名"**
3. **输入您的域名**：`www.0slm.com.cn`
4. **选择"自动添加CNAME记录"**
5. **点击"提交"**

### 第二步：验证域名所有权
1. **按照提示添加TXT记录**
2. **等待验证通过**（通常几分钟）

### 第三步：配置HTTPS证书
1. **阿里云自动申请免费SSL证书**
2. **等待证书签发**（通常几分钟）
3. **开启强制HTTPS**
