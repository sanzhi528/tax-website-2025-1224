# 360软件管家劫持网页链接解决方案

## 问题确认
**问题现象**：点击 `http://financial-tax-website.oss-cn-shenzhen.aliyuncs.com` 链接时，自动跳转到360软件管家进行下载。

**根本原因**：360软件管家劫持了HTTP/HTTPS协议的默认打开方式。

## 解决方案（按优先级排序）

### 🚀 方案1：快速修复默认浏览器设置

#### 方法A：使用批处理工具（推荐）
1. 双击运行：`修复默认浏览器设置.bat`
2. 按照提示设置默认浏览器为Edge或Chrome
3. 重新访问OSS链接

#### 方法B：手动设置
1. 按 `Win + I` 打开设置
2. 进入"应用" → "默认应用"
3. 找到"Web浏览器"，选择"Microsoft Edge"或"Google Chrome"
4. 重新测试OSS链接

### 🔧 方案2：直接修复协议关联

#### 重置HTTP协议关联
1. 按 `Win + R`，输入 `regedit` 打开注册表
2. 导航到：`HKEY_CURRENT_USER\Software\Microsoft\Windows\Shell\Associations\UrlAssociations\http`
3. 删除 `UserChoice` 项
4. 重启电脑

#### 使用命令重置
```cmd
reg delete "HKCU\Software\Microsoft\Windows\Shell\Associations\UrlAssociations\http\UserChoice" /f
reg delete "HKCU\Software\Microsoft\Windows\Shell\Associations\UrlAssociations\https\UserChoice" /f
```

### ⚠️ 方案3：卸载360软件管家（彻底解决）

#### 步骤
1. 打开"控制面板" → "程序和功能"
2. 找到"360软件管家"或相关360产品
3. 右键选择"卸载"
4. 重启电脑

#### 替代方案
- 保留360安全卫士，但禁用其浏览器关联功能
- 在360设置中取消"网页链接保护"或"软件下载关联"

## 验证方法

### 验证修复效果
1. **右键测试**：右键点击OSS链接 → "复制链接地址"
2. **粘贴到浏览器**：将链接粘贴到浏览器地址栏直接访问
3. **直接输入**：在浏览器地址栏手动输入链接

- 不再跳转到360软件管家
- 网站功能完整（登录、咨询、服务展示等）

## 长期预防措施

### 浏览器管理
1. 设置一个主要浏览器为默认（推荐Chrome或Edge）
2. 定期检查默认应用设置
3. 避免安装过多浏览器管理软件

### 安全软件设置
1. 在360设置中禁用"网页链接保护"
2. 取消"软件下载关联"功能
3. 或考虑使用Windows Defender替代

## 技术支持
如果以上方法无效：
- 联系360技术支持要求解除劫持
- 考虑卸载所有360产品
- 使用专业的安全软件替代
