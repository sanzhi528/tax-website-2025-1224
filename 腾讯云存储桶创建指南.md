# 腾讯云对象存储（COS）创建指南

## 一、创建存储桶步骤

### 1. 登录腾讯云控制台
- 访问：https://console.cloud.tencent.com/cos
- 使用腾讯云账号登录

### 2. 创建存储桶
1. 点击左侧菜单"存储桶列表"
2. 点击"创建存储桶"按钮
3. 填写存储桶配置：

**基本信息：**
- **存储桶名称**：`finance-tax-website`（必须全局唯一）
- **所属地域**：**华南地区（广州）**（推荐选择）
- **访问权限**：**公有读私有写**（重要！）

**高级配置：**
- **版本控制**：开启（便于回滚）
- **日志存储**：开启（便于问题排查）
- **静态网站**：开启（必须开启）

### 3. 配置静态网站
1. 进入创建的存储桶
2. 点击左侧"基础配置" → "静态网站"
3. 设置：
   - **索引文档**：`index.html`
   - **错误文档**：`index.html`（单页应用）
   - **强制HTTPS**：开启（推荐）

### 4. 上传网站文件
1. 进入存储桶 → "文件列表"
2. 点击"上传文件"或直接拖拽
3. 上传 `e:\\20251124\\网站建设` 文件夹所有内容
4. 确保文件结构：
   ```
   index.html
   css/style.css
   js/main.js
   images/LOGO图标.jpg
   CNAME
   .nojekyll
   ```

## 二、绑定自定义域名

### 1. 添加自定义域名
1. 进入存储桶 → "域名管理" → "自定义加速域名"
2. 点击"添加域名"
3. 填写：
   - **域名**：`www.0slm.com`
   - **加速地域**：全球加速
   - **源站类型**：默认源站

### 2. 配置CNAME记录
1. 登录DNSPod控制台：https://console.dnspod.cn
2. 找到 `0slm.com` 域名
3. 添加CNAME记录：
   - **主机记录**：`www`
   - **记录类型**：`CNAME`
   - **记录值**：腾讯云提供的CNAME地址
   - **TTL**：600（10分钟）

### 3. 重复步骤绑定其他域名
- `www.0slm.cn`
- `www.0slm.com.cn`

## 六、注意事项

1. **备案要求**：使用国内域名必须完成ICP备案
2. **HTTPS证书**：腾讯云自动提供免费SSL证书
3. **缓存配置**：合理设置缓存时间提升性能
4. **监控告警**：设置流量和请求量监控
5. **备份策略**：定期备份重要文件

## 七、故障排除

### 常见问题
1. **403错误**：检查存储桶权限和文件权限
2. **404错误**：检查文件路径和索引文档设置
3. **域名无法访问**：检查CNAME记录和DNS解析
4. **HTTPS错误**：检查SSL证书状态
