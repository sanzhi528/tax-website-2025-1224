## 🔧 部署步骤

1. 确保所有网站文件完整
2. 主要文件包括：
   - index.html（首页）
   - css/style.css（样式）
   - js/main.js（脚本）
   - 其他HTML页面

### 第二步：注册Vercel账户
1. 访问 https://vercel.com
2. 使用GitHub账户注册（推荐）
3. 完成邮箱验证

### 第三步：部署网站
1. 在Vercel控制台点击"New Project"
2. 选择"Import Git Repository"
3. 上传您的网站文件
4. 等待部署完成

### 第四步：获取临时访问地址
部署完成后，Vercel会提供：
- 临时地址：`your-site.vercel.app`
- 这个地址可以临时访问您的网站

## 🔗 域名绑定流程

### 在域名注册商处设置DNS
1. 登录您的域名注册商（如阿里云、腾讯云等）
2. 找到DNS管理/域名解析设置
3. 添加以下记录：

```
类型：CNAME
主机：www
指向：cname.vercel-dns.com
TTL：3600

类型：A
主机：@（或留空）
指向：76.76.21.21
TTL：3600
```

### 在Vercel添加自定义域名
1. 进入项目设置 → Domains
2. 添加您的域名：www.0slm.com
3. 按照提示验证域名所有权
4. 等待DNS生效（通常需要几分钟到几小时）
